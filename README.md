# Web del Restaurante

Página web simple de un restaurante hecha con HTML y CSS.

## Archivos

- `index.html` - Página principal
- `menu.html` - Página del menú
- `contacto.html` - Página de contacto
- `style.css` - Estilos
- Imágenes de platos del restaurante

## Cómo usar

1. Abre el archivo `index.html` en tu navegador web
2. Listo, ya puedes ver la página web

## Tecnologías

- HTML5
- CSS3
